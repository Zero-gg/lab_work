# Система учета экзаменационной нагрузки (C++)

Проект состоит из двух взаимодействующих программ для работы с текстовыми файлами (`teachers.txt` и `exams.txt`).

## Функционал

### Программа 1: Ввод данных
* Создание и дозапись файлов.
* Строгая валидация ввода (проверка типов данных, дат, форматов).
* Кроссплатформенная очистка консоли.

### Программа 2: Анализ и Отчетность
* **Оптимизация:** Использование метода индексации для работы с файлами произвольного размера без полной загрузки в ОЗУ.
* **Логика:** Реализовано двустороннее сопоставление данных: в отчет попадают как совпадения, так и уникальные записи из обоих файлов с пометкой 'НЕТ ДАННЫХ' для отсутствующих полей.
* **Режимы работы:**
    1. Генерация полного отчета в файл `report.txt` (включая записи без пары).
    2. Избирательный поиск и вывод информации по конкретному предмету на экран.

## Технологии
* ЯП: C/C++ (библиотеки: map, vector, set, string)
* Работа с файлами (FILE*, fseek, ftell)
